The Developer

Hannah Johnson created this site as a Hackbright Academy Software Engineering Program project in May - June of 2019. Reflecting on her past experience as a preK teacher, she realized that licensing and citation info for child care facilities is not especially easy to access and digest. She decided to create a more user-friendly site to improve accessibility. With Child Care Check you can filter by data on adherence to state licensing, location, and facility name/key words.

Child Care Check

This site is built based off of data from the California Department of Social Services. It highlights data on child care centers' adherence to licensing standards within the state of CA. The information on this site was last updated on May 15, 2019. If you need to ensure info you find on this site is up-to-date, please visit the CA state website from which this data was retrieved.

You can find the code for this project on GitHub.
